# libppmp

    libppmp是一个仅头文件的库，为C/C++预处理器元编程提供了基本的实现。<br>
    defs_gen是生成辅助宏的工具，include/ppmp/defs下的文件均由该工具生成，您也可以根据自身需求调整参数重新生成这些宏。<br>

# 许可

    libppmp以LGPL-3.0 Linking Exception许可进行分发，其在原LGPL-3.0许可的基础上移除了您在静态链接时必须提供最小对应源代码的义务。简而言之，只要您不修改本库的源代码，不论您是静态链接还是动态链接，都无需承担开源义务。<br>
