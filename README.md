# libppmp

    libppmp is a header-only library that provides a basic implementation for C/C++ preprocessor metaprogramming.<br>
    defs_gen is a tool for generating auxiliary macros. The files under include/ppmp/defs are all generated by this tool. You may also adjust the parameters according to your own needs to regenerate these macros.<br>
